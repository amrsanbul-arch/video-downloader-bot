# 🆕 آخر التحديثات (Version 2.0)

## ✨ الميزات الجديدة

### 1️⃣ **لوحة إدارة كاملة (Admin Dashboard)**
بدل الأوامر النصية، استخدم أزرار تفاعلية:
```
/admin
```
تظهر لك لوحة بالخيارات التالية:
- 👥 المستخدمين
- 📊 الإحصائيات
- 📢 برودكاست (أرسل رسالة لكل المستخدمين بزر واحد)
- 🚫 حظر/رفع حظر
- 📝 السجلات
- 🔄 تحديث من GitHub
- 🔌 إعادة تشغيل البوت

### 2️⃣ **اختيار جودة دقيقة**
لما تبعت رابط فيديو، البوت الآن يعرضلك خيارات جودة دقيقة:
- 🎥 2160p (4K)
- 🎬 1080p (Full HD)
- 📱 720p (HD)
- 📞 480p (Mobile)
- 🎵 صوت MP3
- 🖼 صورة مصغرة

بدل عالي/متوسط العامة.

### 3️⃣ **نظام Force Subscribe (اختياري)**
تفرض على المستخدمين الاشتراك في قناتك قبل التحميل:

**التفعيل في `.env`:**
```
FORCE_SUBSCRIBE_CHANNEL=channel_name
```
(ضع اسم القناة بدون @)

لو تريد تعطيله، اتركه فاضي.

### 4️⃣ **شريط تقدم (Progress Bar)**
البوت الآن يُحدّث الرسالة وقت التحميل بشريط تقدم:
```
⏳ جاري التحميل...
[████░░░░░░] 40%
```

### 5️⃣ **تحسينات عامة**
- كاش للفيديوهات المحملة مؤخراً
- رسائل خطأ أوضح
- معالجة أفضل للمشاكل

## 📝 التعديلات على الملفات

### ملفات جديدة:
- `handlers/admin_dashboard.py` — لوحة الإدارة الكاملة
- `handlers/force_subscribe.py` — نظام الاشتراك الإجباري

### ملفات محدثة:
- `bot.py` — استيراد الملفات الجديدة
- `handlers/download.py` — تحسين تجربة التحميل مع جودات دقيقة
- `services/downloader.py` — دعم جودات محددة (480p, 720p, إلخ)
- `config.py` — إضافة إعدادات جديدة
- `.env.example` — متغيرات الإعدادات الجديدة

## 🚀 كيفية التحديث

### إذا كان البوت شغال بالفعل:

**1. توقيف البوت:**
```bash
cd ~/storage/downloads/video_bot
python bot.py
```
اضغط `Ctrl+C`

**2. نسخ الملفات الجديدة إلى المجلد**
(ستحصل على ملف `video-downloader-bot-v2.zip` محدث يحتوي على كل الملفات)

**3. فك الضغط والنسخ:**
```bash
unzip video-downloader-bot-v2.zip
cp -r video_bot/* ~/storage/downloads/video_bot/
```

**4. شغّل البوت تاني:**
```bash
cd ~/storage/downloads/video_bot
python bot.py
```

## 📖 أمثلة الاستخدام

### استخدام لوحة الإدارة:
```
/admin
```
ثم اختر العملية اللي تبيها من الأزرار.

### تفعيل Force Subscribe:
```bash
nano .env
```
ضيف:
```
FORCE_SUBSCRIBE_CHANNEL=your_channel_name
```

### استخدام جودات مختلفة:
- ابعت رابط الفيديو
- اختر من الأزرار اللي بتظهر (4K, 1080p, 720p, 480p, MP3, أو صورة)

## 🐛 تصحيح الأخطاء

إذا حصل خطأ:
1. تأكد من أن كل ملفات Python صحيحة الصياغة
2. شغّل البوت مع عرض الأخطاء مباشرة (مش جوه tmux) لترى الخطأ بالظبط
3. شيك على `logs/bot.log` لأي رسائل خطأ
